sfs (Simple File Server)
========================

sfs is a simple file server that will serve a directory on your local file system
via HTTP.

## Usage

```
$ sfs --help                                                                                                                                                                        ~/g/s/g/b/sfs
Usage of sfs:
  -path string
        The path to serve. (default ".")
  -port int
        The port to listen on. (default 6100)
```
